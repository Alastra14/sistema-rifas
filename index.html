3

✨ Características

  • Interfaz intuitiva para configurar rifas sin programar
  • Soporte para múltiples premios con cantidades personalizables
  • Carga de participantes mediante archivo CSV
  • Template de Excel/CSV descargable con formato correcto
  • Preparado para integración con APIs (próximamente)
  • Animaciones profesionales durante el sorteo
  • Exportación automática de configuración en JSON
  • Sistema drag & drop para subir archivos

📁 Formato del CSV
